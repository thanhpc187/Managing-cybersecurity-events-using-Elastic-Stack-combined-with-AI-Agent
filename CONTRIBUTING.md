# Contributing Guidelines

## Welcome!

Thank you for your interest in contributing to this project!

## Attribution

If you use this code, please credit the original author (thanhpc187) and include a link to this repository.

## Contributing Code

### Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request with a clear description

### Code Style

- Follow existing code style
- Add appropriate comments
- Update documentation as needed
- Ensure all tests pass

### License

By contributing, you agree that your contributions will be licensed under the same MIT License.

## Reporting Issues

### Bug Reports

- Use the GitHub issue tracker
- Provide detailed reproduction steps
- Include error messages and logs
- Specify environment details

### Security Issues

- Do NOT use public issue tracker for security issues
- Contact the repository owner directly

## Questions

For questions, please open an issue on GitHub.

---

Thank you for contributing!
